## Problem

需要为 ProjectFlow 添加用户体系功能，目前系统缺少用户管理和认证相关功能。

## Solution

### 需求分析（待确认）

1. **本地账户 + 权限系统**
   - 用户注册/登录，本地 SQLite 存储
   - 角色：管理员、普通用户

2. **多用户联网协作**
   - 类似在线文档的协作模式
   - 数据同步机制

3. **管理员功能**
   - 审核用户项目申请
   - 对项目/文档的读写控制
   - 用户管理（新增/删除）
   - 权限分配（项目分配、任务分配）

4. **用户操作流程**
   - 用户申请新建项目 → 管理员审核 → 赋予权限 → 操作项目

5. **数据同步**
   - 用户数据、项目数据、任务数据的同步

### 待讨论问题

- 联网模式：P2P？中心化服务器？
- 离线支持：是否需要离线工作能力？
- 同步策略：实时同步 vs 定时同步？
