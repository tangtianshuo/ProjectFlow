## Problem

需要为 ProjectFlow 添加 LLM 辅助功能，让用户通过对话方式编写项目文档。

## 需求（已确认）

### 1. 多模型支持
- 使用第三方库实现
- 支持多种 LLM 提供商

### 2. 对话功能
- 通过新模块实现（独立于文档编辑）
- 独立菜单项："AI 助手"

### 3. 文档类型
- 支持所有文档类型

### 4. 功能范围
- 文档生成
- 回答关于项目的问题
- 分析现有文档并提出建议

### 5. API Key 管理
- 用户自行配置
- 加密存储

### 6. 上下文理解
- 自动将当前项目信息注入对话
