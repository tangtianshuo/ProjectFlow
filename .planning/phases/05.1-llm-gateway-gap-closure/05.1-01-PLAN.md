# llm-gateway API (from crate documentation)
llm-gateway provides a simple API:
```rust
use llm_gateway::Gateway;

// Create gateway with API key
let gateway = Gateway::new("openai", "api-key").await?;

// Chat completion
let response = gateway.chat().model("gpt-4").messages(messages).await?;

// Stream
let mut stream = gateway.chat().model("gpt-4").messages(messages).stream().await?;
```
