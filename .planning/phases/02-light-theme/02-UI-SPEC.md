## Theme Switching (LIGHT-04)

**Source:** New requirement for Phase 2

| Feature | Implementation |
|---------|----------------|
| Toggle location | Settings page or sidebar |
| Toggle component | Icon button (sun/moon icons) |
| Persistence | LocalStorage key: "projectflow-theme" |
| Default | Respect system preference, fallback to dark |
| Transition | 200ms ease-in-out on all color changes |

**CSS Implementation Pattern:**

```css
[data-theme="light"] {
  --bg-primary: #fafafa;
  --bg-secondary: #ffffff;
  --text-primary: #18181b;
  --text-secondary: #52525b;
  /* ... all light theme tokens */
}

[data-theme="dark"] {
  --bg-primary: #08080c;
  --bg-secondary: #0f0f14;
  --text-primary: #f4f4f5;
  --text-secondary: #a1a1aa;
  /* ... all dark theme tokens */
}
```

**Vue Store Extension:**

```typescript
// In uiStore.ts, add:
const theme = ref<'dark' | 'light'>('dark');

function toggleTheme() {
  theme.value = theme.value === 'dark' ? 'light' : 'dark';
  document.documentElement.setAttribute('data-theme', theme.value);
  localStorage.setItem('projectflow-theme', theme.value);
}
```
